## K-arry FizzBuzz 
*Author: Lee-Roy King*

---

### Problem Domain
***[Problem Domain for the Code Challenge]***

Conduct “FizzBuzz” on a k-ary tree while traversing through it.
Set the values of each of the nodes depending on the corresponding node value in the source tree.
